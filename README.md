# 项目描述
本项目使用Gin+Vue+Echarts+MySQL+Redis+随机森林算法等技术栈，采用Viper、JWT、Zap、Swagger、雪花算法等中间件进行开发，对冬奥会数据进行可视化分析展示并预测各个国家在下一届冬奥会的奖牌数量，适合没有Go项目经验的小伙伴进行开发。
# 使用指南
### 1. 配置config.yaml
配置MySQL、Redis数据库和发送邮件信息。
### 2. 运行项目
`go mod tidy`

`go build`

`go run main.go`
